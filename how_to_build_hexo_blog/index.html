用 hexo 建立一個部落格

簡介

hexo 是一個可以使用markdown 語法產生靜態頁面的部落格工具。
可以配合 github 快速建立一個部落格。

安裝環境：linux(ubuntu12.4)

安裝 nvm 和 hexo

hexo n
hexo new [layout] [title]

hexo g (= hexo generate)

用於建造網頁的時候，hexo 會將 markdown 格式編譯成 html 格式

hexo g —watch

使用此命令後，重開 hexo，可以使 hexo 偵測檔案變更，並自動 generate

hexo s (= hexo server)

在本地預覽網站 - 預設在http://localhost:4000/

hexo s -d

在預覽的時候同時看到在草稿的文章

hexo d (hexo deploy)

發佈網站至設定的網址，網站會真正更新

hexo g -d

先 generate 再 deploy

hexo publish [filename] (= hexo p [filename])

將檔案 (從草稿) 移至 /post

需先備份設定檔_config.yml

修改主題

自行修改主題後, 要先清除原有主題，style.css 才會正確

將留言區改成 disqus

http://mwshen.info/2014/11/14/Hexo-pacman%E4%B8%BB%E9%A2%98%E5%90%AF%E7%94%A8disqus/

回到頂部功能

http://wsgzao.github.io/post/hexo-scroll-to-top/

數學公式

使用 mathjax(適用 tex 語法)
http://catx.me/2014/03/09/hexo-mathjax-plugin/

需要注意的是若有兩個底線 _，會變斜體，這是原有的 markdown 格式
解決方法：用
{
%
math-block
%
}
和
{
%
endmath-block
%
}
包覆數學公式

小缺點是會使網頁載入速度變慢(公式的圖片是即時取得的)
