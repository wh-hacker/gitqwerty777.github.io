Emacs常用指令表

基本

  • C → Ctrl
  • M → Alt or escape
  • S → Shift
  • [Ret] → return(enter)
  • 例 C-c → Ctrl + c
  • 「hello.c」的自動儲存檔案的檔名就是「#hello.c#」。儲存檔案後，自動儲存檔就會被刪除
  • Ctrl 通常是 編輯/移動 一個字， Alt 通常是 編輯/移動 一個詞

移動

指令 介紹
M-v, C-v 前翻/後翻一頁
M-b, M-f 往前/往後 移動一個詞
M-a, M-e 移動到一句的開始/結尾
C-l 移動視窗(遊標分別在中間、最上方、最下方)
M-g M-g 跳至指定行(= M-g g)

編輯

指令 介紹
C-j 換行並自動縮排(newline-and-indent)
C-x h 全選
C-SPC 選取文字(再按一次結束選取)
C-w, M-w 剪下/複製 選取文字
C-y 貼上選取文字
M-y 選擇貼上的文字(從之前剪下/複製過的文字中)
C-d 同delete鍵(刪除後一個字)
M-d 刪除後一個詞
C-g 取消尚未完成的指令

C-k : 刪除從游標到「行尾」間的字元
單獨的 C-k 會把一行的內容殺掉
而第二個 C-k 則會殺掉那一行本身(‘\n’)。使所有其他的行向上移動

C-x u : 還原
(1) 沒有改變文字的指令不算（移動游標和視窗等）
(2) 自行鍵入的字元 一次最多還原20個

重複

指令 介紹 示範
C-u [數字] [指令or字元] 重複執行 C-u 8 a → aaaaaaaa

搜尋

指令 介紹
C-s 往後搜尋(再按一次：找下一個)
C-r 往前搜尋
M-C-s 正規表示式搜尋
M-% ! 取代
  • y：置換目前游標所在的字串，並且移到下一個可被置換的字串。
  • n：跳過目前游標所在的字串。移到下一個可被置換的字串。
  • !：將整個檔案一次置換完畢，不進行任何詢問。
  • ESC：結束置換模式。
  • .：將目前的字串置換後，即離開置換模式。
  • ^：回到前一個符合置換的地方。

分割

指令 介紹
C-x 0 關閉目前分割
C-x 1 關閉其他分割(只留下目前的分割)
C-x 2 新增水平分割
C-x 3 新增垂直分割
C-x o 跳到別的分割
C-x 5 2 開新視窗(frame)

檔案

檔案切換

指令 介紹
C-x C-b 列出檔案清單(Buffer List)
C-x b 切換至其他開啟的檔案
C-x k 關閉目前編輯的檔案，不存檔

檔案指令

指令 介紹
C-x C-f 開新檔案，開啟舊檔

C-x C-w 另存新檔
C-x d 開啟資料夾(Dired)

系統

指令 介紹
C-x C-c 結束 Emacs
C-z 暫時離開Emacs(Suspend)

用fg或%emacs恢復 Emacs

函式

指令 介紹
M-x [function] 執行名為[function]的功能
M-x compile 開啟compilation分割進行compile
M-x recompile 重覆執行上次執行過的compile

如 C-c C-c 為中止程式; C-c C-z 為暫停程式等

幫助

C-h i 開啟 info

rot13

rot13 是一種很簡單的凱撒位移加密方法，普遍用於各種迷題後面所附的答案等 的加密上，目的是讓你並不能一下子「不小心」看到了答案，同時又讓你在需要知 道答案的時候能夠方便地進行解密。

rot13 的加密和解密的方法都非常簡單，就是把每個字母往後移動 13 個即可。 因為剛好二十六個字母，所以加密和解密都使用同樣的方法。雖然也可以採用手 工加解密的方法，但是自動完成通常更方便。在 UNIX 系統上，可以直接用 tr 實用程序來完成這個工作：

tr A-Za-z N-ZA-Mn-za-m

大多數新聞閱讀器都帶有自動加解密的功能，Emacs 也有一個 rot13-region 的 命令可以方便地使用。

不過，當然，這種只對英文字母進行轉換的加密，對於中文來說沒有任何作用呢。
