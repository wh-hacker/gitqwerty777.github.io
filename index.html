Emacs常用指令表

基本

  • C → Ctrl
  • M → Alt or escape
  • S → Shift
  • [Ret] → return(enter)
  • 例 C-c → Ctrl + c
  • 「hello.c」的自動儲存檔案的檔名就是「#hello.c#」。儲存檔案後，自動儲存檔就會被刪除
  • Ctrl 通常是 編輯/移動 一個字， Alt 通常是 編輯/移動 一個詞

經濟學一

Introduction

經濟學是什麼？

經濟學是理解「人的行為」的方法：
個人從事行為前會衡量利弊得失，有一定脈絡可循
個人的行為反應其自利動機，人為自己而活
社會由個人組成，從個人行為出發，解釋各種社會現象

經濟學家的眼鏡：將每個人視為理性、自利
