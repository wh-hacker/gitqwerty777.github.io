大數運算的函式庫—BigNumber.h

因為在許多 online judge 的題目中，大數運算是很常見的題型之一， 所以就寫了一個可以簡單進行大數運算的函式庫。

內容

目前只支援整數
四則運算 (除法只算到整數)
可直接比較，賦值
從各種 type 轉換成 class BigNumber，如 int, long, string…
以 string 形式輸出
