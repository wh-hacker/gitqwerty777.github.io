在 Linux 上使用 fcitx 安裝嘸蝦米

之前用過 gcin, scim 等輸入工具，發現對嘸蝦米的支援十分有限。fcitx 預設即有嘸蝦米碼表，較為方便，安裝也比較簡單。

安裝 fcitx 輸入工具

sudo apt-get install im-switch fcitx

安裝嘸蝦米於 fcitx

sudo apt-get install fcitx-boshiamy

設為預設值

使用 im-switch 將 fcitx 設為預設輸入法
sudo im-switch -s fcitx

調整 fcitx 設定：fcitx 設定 -> 附加組件配置 -> 碼表 ->boshiamy.config

模糊鍵

『使用模糊鍵』不勾
12.04 版 - 此鍵預設被打勾，並定為 z 鍵，導致所有含 z 碼的字都出不來

使用符號

要用嘸蝦米預設符號:「忽略標點」打勾

拼音查蝦米碼

在拼音鍵打勾，最好選 `（Esc 下的那鍵），才不影響輸入，然後用 -, = 兩鍵翻頁。
（至於輸入的字要查發音則是屬於另一個功能，在剛按完空白鍵輸入後，就按 Ctrl+Alt+E，即可看見發音，但缺點是沒有四聲）
