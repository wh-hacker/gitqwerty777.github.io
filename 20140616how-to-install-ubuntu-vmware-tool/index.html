在Ubuntu虛擬機上安裝vmware_tool

VMware-tools是Vmware提供，用來增強虛擬顯卡和硬盤性能、以及同步虛擬機與主機時鐘的驅動程序

另解

sudo apt-get install open-vm-tools

選擇性安裝 

sudo apt-get install open-vm-dkms   
sudo apt-get install open-vm-toolbox
