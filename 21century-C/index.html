連結函式庫

  • 告知兩次
    • #include
    • 編譯器 -l
  • Example
    • #include <math.h>
    • -lm
  • 標準函式庫不用連結
    • 編譯時隱含-lc，連結到libc

  • 路徑

    • 預設為標準函式庫路徑
    • include檔路徑 I[path]
    • 函式庫路徑 -L[path]
    • 按照相依順序列出
      • -lbroad -lgeneral
      • 在board未對應的函數會在general尋找

    • pkg-config

      • 回傳套件的-I和-l, -L參數

    • 用backtick括起來，會用命令執行的結果取代命令

      • pkg-config --libs openssl = -lssl -lcrypto

函式庫類型

  • 靜態函式庫(static library)
    • 編譯器連結
  • 共享函式庫(shared library)
    • 執行期連結
    • 解法
      1. 用Libtool打包
      2. 設定LD_LIBRARY_PATH環境變數
      3. -Wl, -R[PATH] 將-L傳送到linker, -R嵌到執行期
