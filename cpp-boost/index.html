C++ boost 簡單應用

簡介

C++的著名擴充函式庫
功能(節錄)：

  • regex
  • function binding
  • lambda functions
  • unit tests
  • smart pointers
  • noncopyable, optional
  • serialization
  • generic dates
  • portable filesystem
  • circular buffers
  • config utils
  • generic image library
  • program options
  • threads

http://www.boost.org/doc/libs/1_58_0/libs/libraries.htm

安裝：sudo apt-get install libboost-all-dev

shared pointer

boost::shared_ptr指針sp1和sp2同時擁有了implementation對象的訪問權限，且當sp1和sp2都釋放對該對象的所有權時，其所管理的的對象的內存才被自動釋放。在共享對象的訪問權限同時，也實現了其內存的自動管理。
